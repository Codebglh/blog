# Mac 修改 pip 镜像源

  1. 进入根目录： cd ~/
  2. 进入.pip 目录 cd .pip
  3. 如果不存在文件夹就新建 mkdir .pip
  4. 进入 cd .pip
  5. 创建 pip.conf 文件 touch pip.conf
  6. 修改： vim pip.conf
